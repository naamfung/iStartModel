# iStartModel
LlamaServer大语言模型引擎的启动器，配置集中管理..

# 经验
通常模型内置的官方模板在驱动智能体时或多或少都有些问题，大抵系由于开发团队测试的环境未有大家真实环境那样多样化导致，如系千问 3.5 / 3.6 模型或以其作为基座的衍生模型请优先使用 tmpl 目录中经过真实智能体工作环境考验的 Qwen-Agentic-EN / Qwen-Agentic-HON(S/T) 模板文件，或使用模板目录的froggeric-v21.3千问系列模板。
